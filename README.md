<h2>Como Utilizar</h2>
<p>

```

- Clone o repositório:
$ git clone [https://github.com/DuhAlonso/picpay_clone.git](https://github.com/RegisSalesRA/Flutter_Hive.git)

- Entre no diretório:
$ cd flutter_hive

- Instale as dependências:
$ flutter pub get / flutter packages get

- Execute:
$ flutter run

```
